# 🎮 MandaGame

Game petualangan 2D berbasis HTML5 yang dibuat menggunakan Construct 2.
Pemain berperan sebagai seorang penjelajah yang harus mengumpulkan kunci dan mencapai tujuan akhir.

---

## 🕹️ Gameplay

* Gerakkan karakter menggunakan keyboard (arrow / WASD)
* Kumpulkan minimal **5 kunci**
* Setelah terkumpul, menuju titik akhir untuk menang

## 📁 Struktur Project

* `index.html` → entry point game
* `c2runtime.js` → engine runtime
* `data.js` → data game
* `images/` → asset visual
* `media/` → audio
